# 🧮 Cálculo Mental

Juego de cálculo mental desarrollado con React Native y Expo.

## Modos de juego

- **Clásico** — Ingresá el resultado con el teclado numérico
- **Verdadero / Falso** — Indicá si el resultado propuesto es correcto
- **Múltiple Choice** — Elegí entre 4 opciones
- **Contra Reloj** — Respondé todo lo que puedas en 60 segundos

## Dificultades

| Nivel | Tiempo | Números | Operaciones |
|-------|--------|---------|-------------|
| Fácil | 15s | 1 – 20 | +, - |
| Medio | 10s | 1 – 50 | +, -, × |
| Difícil | 6s | 1 – 100 | +, -, ×, ÷ |
